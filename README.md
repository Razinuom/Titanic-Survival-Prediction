# Titanic Survival Prediction

A machine learning project that predicts whether a Titanic passenger survived based on features like age, sex, and passenger class.

Built as a beginner ML project to learn the end-to-end data science workflow.

## What I did
- Explored and cleaned the Titanic dataset from Kaggle
- Engineered features and handled missing values
- Built a Random Forest classifier using scikit-learn
- Used a Pipeline to chain preprocessing and modelling steps
- Tuned hyperparameters with GridSearchCV to improve accuracy
- Generated predictions on the test set

## Libraries used
- pandas, numpy, matplotlib
- scikit-learn

## Dataset
From the [Kaggle Titanic Competition](https://www.kaggle.com/competitions/titanic)
